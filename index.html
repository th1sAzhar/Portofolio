About Me

Halo! Saya Azhar Indrawan, seorang IT Enthusiast dan kreator digital yang berdedikasi tinggi dalam pengembangan game interaktif serta perancangan pengalaman pengguna (UI/UX). Saya memiliki ketertarikan besar dalam menjembatani kreativitas visual dengan logika pemrograman.

Sejauh ini, saya telah berpengalaman membangun game 2D Side Scroller menggunakan engine Unity (C#), serta merancang arsitektur aplikasi mobile untuk efisiensi bisnis lokal melalui program Kampus Merdeka Entrepreneur. Di luar coding, saya mengasah kemampuan kepemimpinan dan manajemen sebagai Community Manager di server Discord FiveM, sekaligus mendalami otomatisasi sistem lewat pengembangan Discord BOT dan pengelolaan infrastruktur VPS Linux.
